# connect4-ai
## If you want to modify the game do this steps.
1. Install python (I recommend using `winget`).
```
winget install --id=Python.Python.3  -e
```
2. Install dependencies.
```cmd
pip install numpy
pip install pygame
```
3. Clone Repo.
```
git clone https://github.com/Blitherium/connect4-ai.git 
cd connect4-ai
```
4. Run File.
```
python mainmenu.py
# or
py mainmenu.py
```
